# 🏡 AI House Price Prediction

A Machine Learning web application that predicts house prices using Linear Regression. The project is built using Python, Flask, Scikit-learn, HTML, and CSS.

## 🚀 Features

- Predicts house prices instantly
- Machine Learning model using Linear Regression
- Flask-based web application
- Responsive and modern user interface
- Easy-to-use web form

## 🛠 Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML5
- CSS3

## 📁 Project Structure

```
House_Price_Prediction/
│── app.py
│── model.py
│── house_model.pkl
│── requirements.txt
│── README.md
│── dataset/
│── static/
│── templates/
```

## ▶️ How to Run

```bash
pip install -r requirements.txt
python model.py
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

## 📷 Output

- User enters house details
- Model predicts estimated house price
- Result is displayed on the webpage
